# franka_visualizer
Visualize franka arm FK and IK.

## installation
```bash
conda create -n franka_visualizer python=3.8
conda activate franka_visualizer
pip install -r requirements.txt
```

## usage 
- Run 'python main.py', and follow the GUI instruction.
